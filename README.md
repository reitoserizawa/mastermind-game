<h2>How to Play</h2>
<p>1. Clone this repository</p>

```
git clone git@github.com:reitoserizawa/mastermind-game.git
```

<p>2. Run Python scripts for the 'mastermind-game-oop' folder</p>

```
python3 mastermind-game-oop
```

<p>3. Enjoy the game!</p>
